Self-host Google Analytics v4 or use Minimal Analytics, a lightweight gtag.js alternative, to improve site speed and enhance privacy.

== Description ==

Host Google Analytics locally using any one of these:

- Gtag.js (90KB)
- Minimal Analytics.js (1.4KB)

Optionally, disable Analytics when admin is logged in.

== Installation ==

1. Visit 'Plugins > Add New'
1. Search for 'Flying Analytics'
1. Activate Flying Analytics for WordPress from your Plugins page.
1. Visit 'Settings -> Flying Analytics' to configure
